# Query & Visualization Examples

## Notebooks

There are currently 2 notebooks in this folder:
- CHEFS_SQL_examples.ipynb : This notebook contains SQL queries to the EFSA database we created regarding an example use-case. This is to provide an idea how the database structure works in practice.
- CHEFS_data_visualization.ipynb : This notebook contains Python code to re-create the figures in the article, and it can be used as a reference example of how the data can be visualized.

## Installation instructions
Please follow the instructions in the main folder before performing any query or visualization.\
\
Using the python virtual environment we already created for step 8 in the installation steps in the main README, install the requirements for in this folder:
- `.\venv\Scripts\activate`
- `pip install -r visualization_requirements.txt`
